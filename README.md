# AI-Health-Symptom-Checker
The AI Health Symptom Checker is a Python-based healthcare application built using Streamlit, SQLite, and machine learning techniques. It allows users to enter symptom severity and receive AI-based disease predictions along with confidence scores and health risk analysis. The system also includes BMI calculation and multilingual support for better accessibility.

A Random Forest machine learning model is used to predict diseases such as COVID-19, Flu, Cold, and Viral Fever. The application provides interactive charts, health recommendations, and stores user prediction history in a database. An admin dashboard is also included to analyze disease trends and risk distributions.

Overall, this project demonstrates how artificial intelligence can improve healthcare accessibility and early symptom analysis. It can be further enhanced with real-time medical datasets, advanced AI models, and doctor consultation features.
